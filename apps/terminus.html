The Terminal That Learns How You Work

A next-generation macOS terminal combining traditional shell power with intelligent command assistance. Natural language commands, multi-panel workspace, command safety analysis, learning-based predictions, and semantic search. Works fully offline — AI is optional.

Features

💬

Natural Language Commands

Type "find large files here" and get executable shell commands via AI. OpenRouter integration for intelligent command translation.

▦

Multi-Panel Workspace

Split horizontally and vertically, tabs, spatial keyboard navigation. Full VT100/xterm compatible terminal emulator with true color.

🧠

Learning-Based Predictions

Learns your command patterns, suggests based on frequency, recency, directory context, and project awareness.

⚠

Command Safety Analysis

Detects destructive commands before execution, warns about risks. Prevents accidental damage to your system.

📁

File Explorer

Built-in directory browser with file preview, breadcrumb navigation, and seamless integration with the terminal.

📈

System Monitor

Live CPU, RAM, GPU, disk, and network metrics in a side panel. JSON tree visualization, colored diffs, highlighted errors.
