Brain OS

AI-Powered Desktop Agent

Describe a task in natural language. The agent plans it, you approve it, it executes it. Uses OpenRouter LLMs for reasoning and planning, then executes actions through shell commands and AppleScript — step by step, with full visibility and safety controls.

Features

🧠

Agent Execution Engine

Sequential step-by-step execution with 10 step types: shell, AppleScript, file I/O, LLM reasoning, verification, and more. Automatic error recovery.

🛡

Safety & Transparency

Risk classification engine (low/medium/high). Safe Mode requires approval for risky actions. Pattern analysis detects rm -rf, sudo, etc.

💡

LLM Planning

OpenRouter integration with any model. Structured JSON plan generation — task interpretation, assumptions, risk assessment, steps. Step repair on failure.

📄

Workflows & History

Save successful executions as reusable workflows. Favorite, search, filter, and re-run workflows with one click. Full session logging.

▶

Live Execution Timeline

Pause, resume, cancel mid-execution. See live output, errors, retries, and final result for every step in real time.

💻

macOS Native Integration

Shell commands and AppleScript for deep macOS automation. Permission status panel. 33 Swift files, 11 modules, zero dependencies.
