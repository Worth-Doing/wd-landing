DropVault

Capture Everything. Lose Nothing.

The visual memory layer for macOS. A next-generation clipboard and drag-and-drop command center with menu bar access, floating drop zone, shelves, tagging, sorting, grid/list views, export/import, and full keyboard shortcuts.

 macOS 14+ Swift 5.9 Local-First v2.0.0
Download DMG (2.2 MB) View Source
● Capabilities

Features

📋

Clipboard Monitoring

Automatic clipboard capture with toggle control. Pause/resume from toolbar, menu bar, or settings. Copy confirmation toast.

🗀

Floating Drop Zone

Drag and drop anything onto the floating window. Files, text, images, links — all captured and organized automatically.

📁

Shelves & Tags

Organize items into shelves with custom icons (13 options). Add and remove tags freely. Rename and manage via context menu.

▦

Grid & List Views

Toggle between visual card grid and compact list. 6 sort modes: newest, oldest, A–Z, Z–A, largest first, by category.

🔃

Export & Import

Export your full vault to JSON, import with automatic deduplication. Open links in browser, reveal files in Finder.

☰

Menu Bar Extra

Quick-access menu bar icon with recent items, monitoring toggle, and drop zone control. Always one click away.
