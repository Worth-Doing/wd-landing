Morphix

Local-First File Conversion Studio

The most powerful local-first file conversion studio for macOS. Drop anything — Markdown, HTML, URLs, HEIC, PNG, JPG, WebP, TIFF — and get the right format out. Capability-driven adapters, actor-based job queue, and persistent history. Built with Swift + SwiftUI, no Xcode required.

Features

📄

Markdown / HTML / Text → PDF

Convert .md, .txt, and .html into pixel-perfect PDFs using swift-markdown and WKWebView rendering.

🌐

Webpage URL → PDF

Paste any http/https URL and Morphix fetches it, renders it with WKWebView, and saves a clean PDF.

🎨

Image Conversion

HEIC, JPG, PNG, WebP, TIFF, GIF, BMP — convert between any of them using ImageIO with metadata preservation.

⏩

Remote Media URLs

Drop a remote media URL. HEAD-based content-type classification picks the right adapter automatically.

⚙

Actor-Based Job Queue

Swift actor-backed queue with concurrency, cancel, retry, and event streams. Reveal-in-Finder when done.

💾

Persistent History & Presets

Saved conversion recipes, job history, and settings. Capability-driven target pickers — the UI adapts to what each adapter supports.
