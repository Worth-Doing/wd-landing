OpenRouter Playground

The Ultimate OpenRouter Studio

A professional-grade native macOS playground for OpenRouter. Browse hundreds of AI models, craft multi-turn conversations, tune every generation parameter, stream responses in real-time, compare models side-by-side, and inspect every detail of the API exchange.

Features

💬

Multi-Turn Chat

System, User, Assistant, Developer messages. Add, remove, reorder, collapse. Role switching on the fly with full conversation control.

📚

Model Catalog

Browse all OpenRouter models. Search, filter by provider, price, context length, capabilities. Sort and favorite models.

⚡

Real-Time Streaming

Token-by-token streaming with elapsed timer, chunk counter, live cancel, and smooth rendering via Server-Sent Events.

📈

Model Comparison

Side-by-side comparison (2-up, 3-up, full-width). Same prompt to multiple models with color-coded metrics.

🔍

Response Inspector

5 tabs: Output, Raw, Usage, Timeline, JSON. Full request/response pair with copy and export capabilities.

🎧

Generation Controls

Temperature, Top-P, Max Tokens, Frequency/Presence Penalty, Seed, JSON mode, Stop sequences. Saveable parameter presets.
