Features

☰

Kanban View

Classic column-based board with drag-and-drop cards. Customizable columns, card colors, labels, and priorities.

🔌

Flow Map View

Visualize task dependencies and workflows as a connected flow diagram. See the big picture of your project.

📅

Calendar View

See tasks on a calendar timeline. Due dates, scheduling, and time-based project planning.

▦

Table View

Spreadsheet-like table for bulk editing and sorting. All task properties visible and editable in rows and columns.

📦

Unlimited Boards

Create as many boards as you need. Each board supports all four views. Local storage, instant performance.

🎨

100% Native

No Electron, no WebView, no AI dependencies. Pure Swift + SwiftUI. Feels like a first-class Mac product.
