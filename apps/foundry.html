Foundry

The Native Claude Code Interface

A fully native macOS application that transforms Claude Code from a CLI tool into a visual development studio. Session timelines, usage analytics, skills browser, agents management, and MCP server configuration — all in a premium SwiftUI interface.

 macOS 14+ Swift 6.0 11,000+ LOC v4.0.0
Download DMG (2.9 MB) View Source
● Capabilities

Features

⏱

Session Timeline

Visual timeline with code blocks, tool events, agent spawns, and real-time status tracking for every Claude Code session.

📈

Usage & Cost Analytics

Detailed usage dashboard with cost tracking, token breakdown, and per-session analytics tables.

⚡

Skills Browser

Browse built-in and marketplace plugins with one-click installation. Discover and manage all available skills.

🤖

Agent Management

View built-in agents and configure custom ones. Full agent lifecycle management from a native interface.

🔌

MCP Server Config

Add, configure, and remove MCP (Model Context Protocol) servers. Visual configuration replaces JSON editing.

🎨

Premium Glass UI

Beautiful glassmorphic interface built entirely in SwiftUI. Dark mode, smooth animations, native macOS feel.
