## What is This?

This is the official **Worth Doing app showcase and download portal** — a static website that presents all 10 native macOS applications built by [Worth Doing](https://github.com/Worth-Doing), with direct DMG download links, feature descriptions, screenshots, and installation instructions.

The site is built with **pure HTML, CSS, and JavaScript** — no framework, no build step, no dependencies. It deploys via **GitHub Pages** directly from the `main` branch.

## Design System

The site uses a custom **glassmorphism design system** built from scratch:

| Element | Implementation |
|---------|---------------|
| **Background** | Dark base (`#08090d`) with subtle CSS grid pattern and colored blur glows |
| **Cards** | `backdrop-filter: blur(10px)` with 3–5% white background, 8% white borders |
| **Hover states** | Border brightens to 15% white, card lifts 4px with glow shadow |
| **Typography** | Inter font, -0.04em letter-spacing on headings, 800 weight |
| **Colors** | Indigo accent (`#6366f1`), purple/cyan gradients, semantic colors for tags |
| **Animations** | `fadeInUp` on hero elements, CSS transitions on all interactive elements |
| **Navigation** | Fixed floating nav bar with blur backdrop, mobile-responsive hamburger |
| **Responsive** | CSS Grid with `auto-fill` and `minmax()`, mobile breakpoints at 768px/900px |

## Deployment

The site is deployed via **GitHub Pages** from the `main` branch root.

### Manual Local Preview

```bash
# Clone
git clone https://github.com/Worth-Doing/wd-landing.git
cd wd-landing

# Serve locally (any static server works)
python3 -m http.server 8000
# or
npx serve .
# or just open index.html in a browser
open index.html
```

No build step, no `npm install`, no dependencies. Every page is self-contained HTML linking to the shared CSS and JS files.

### Assets

All app icons and screenshots are loaded from their respective GitHub repositories via `raw.githubusercontent.com` URLs. This means:

- **No asset duplication** — icons and screenshots stay in sync with each app repo
- **No build pipeline** — just push HTML and it works
- **Automatic updates** — when an app updates its screenshots, the site reflects it
